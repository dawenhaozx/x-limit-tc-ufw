# 一键安装
在安装前请确保你的系统支持`bash`环境,且系统网络正常  

```
wget -O lspeed.sh "https://raw.githubusercontent.com/dawenhaozx/x-limit-tc-ufw/main/lspeed.sh" && chmod +x lspeed.sh && ./lspeed.sh
```
